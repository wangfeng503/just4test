# 公考AI助手

行测智能解析 + 申论智能批改 + 企业架构分析 + 领导结构分析

## 功能

- **行测助手**：支持言语理解、数量关系、判断推理、资料分析、常识判断五大题型智能解析
- **申论助手**：申论答案智能批改评分 + 范文生成
- **企业架构**：国企/央企/事业单位组织架构分析
- **领导结构**：领导班子配置分析 + 晋升路径规划

## 使用

1. 打开 https://wangfeng503.github.io/just4test/
2. 首次使用需设置 API Key（支持 DeepSeek / 硅基流动 / OpenAI）
3. 选择功能模块开始使用

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- 调用大模型 API 实现智能分析
- 支持 GitHub Pages 部署
